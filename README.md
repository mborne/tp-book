# tp-book

## Description

Test de jersey & jetty dans le cadre d'un TP de création d'une API.

## Utilisation

* Construction du war : `mvn clean package`

* Exécution avec jetty : `mvn jetty:run`

* Appel API : `curl "http://localhost:8080/tp-book/book/meuh"`
